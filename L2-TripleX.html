Страница как черновик для тестирования задач на ресурсе codeWars

Назавание задачи: L2:TripleX

Сложность: 7kyu

Для данной строки верните истину, если за первым экземпляром «x» в строке сразу следует строка «xx».

Example:

tripleX("abraxxxas") → true

tripleX("xoxotrololololololoxxx") → false

tripleX("softX kitty, warm kitty, xxxxx") → true

tripleX("softx kitty, warm kitty, xxxxx") → false

  • заглавные Х не считаются появлением "х"
  • если нет "х", верните false
