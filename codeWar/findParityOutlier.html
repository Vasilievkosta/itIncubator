Назавание задачи: Find The Parity Outlier

Сложность: 6kyu

Вам предоставляется массив (который будет иметь длину не менее 3, но может быть очень большим), содержащий целые числа.

Массив либо полностью состоит из нечетных целых чисел, либо полностью состоит из четных целых чисел, за исключением одного целого числа N.

Напишите метод, который принимает массив в качестве аргумента и возвращает этот «выброс» N.

Example:

[2, 4, 0, 100, 4, 11, 2602, 36] - Should return: 11 (the only odd number)

[160, 3, 1719, 19, 11, 13, -21] - Should return: 160 (the only even number)
