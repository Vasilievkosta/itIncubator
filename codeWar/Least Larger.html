Страница как черновик для тестирования задач на ресурсе codeWars

Назавание задачи: Least Larger

Сложность: 7kyu

Имеется массив чисел и заданный индекс, верните индекс наименьшего числа, большего чем элемент в заданном индексе, или -1, если такого индекса нет

Example:

leastLarget( [4, 1, 3, 5, 6], 0 ) => 3

leastLarget( [4, 1, 3, 5, 6], 4 ) => -1
