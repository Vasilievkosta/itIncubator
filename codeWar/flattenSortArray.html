Страница как черновик для тестирования задач на ресурсе codeWars

Назавание задачи: Flatten and sort an array

Сложность: 7kyu

Учитывая двумерный массив целых чисел, верните сплющенную версию массива со всеми целыми числами в отсортированном (возрастающем) порядке.

Example:

[[3, 2, 1], [4, 6, 5], [], [9, 7, 8]] => [1, 2, 3, 4, 5, 6, 7, 8 , 9]
