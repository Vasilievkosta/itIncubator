Страница как черновик для тестирования задач на ресурсе codeWars

Назавание задачи: Persistent Bugger.

Сложность: 6kyu

Напишите функцию, persistence которая принимает положительный параметр num и возвращает его мультипликативную постоянство, то есть количество раз, которое вы должны умножать цифры, num пока не дойдете до единственной цифры.

Example:

persistence(39) === 3 // because 3*9 = 27, 2*7 = 14, 1*4=4 and 4 has only one digit

persistence(999) === 4 // because 9*9*9 = 729, 7*2*9 = 126, 1*2*6 = 12, and finally 1*2 = 2

persistence(4) === 0 // because 4 is already a one-digit number
